# 文件介绍
 该工程为 FIC628 MDK,拥有**三种**编译方式：
	

 - cmd_MDK   表示的是，使用命令行方式编译生成bin文件；
 - keil3_MDK  表示的是，使用 keil3 编译方式生成bin文件；
 - keil5_MDK  表示的是，使用 keil5 编译方式生成bin文件；
  
 ## cmd_MDK
 使用命令行编译的所需编译文件
 
## keil3_MDK
使用keil3编译的所需编译文件

## keil5_MDK
使用keil5编译的所需编译文件
	
## Code
保存FIC628编译的所需的代码
	
## doc
该文件保存工程相关文档

## tools
该文件夹保存这个是工程中可能使用的技术类或支持类相关文档；
	

## .gitignore
git 上传所要屏蔽的文件

## all_clear.bat
该文件是用来清除编译生成的文件
